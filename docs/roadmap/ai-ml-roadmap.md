# 🤖 AI / ML 学习路线

> AI 与机器学习方向的体系化学习路径。

---

## 📖 数学基础（1-2 个月）

| 科目 | 重点内容 | 推荐资源 |
|------|---------|---------|
| **线性代数** | 矩阵运算、特征值分解、SVD | 3Blue1Brown 线性代数本质 |
| **概率论与统计** | 概率分布、贝叶斯定理、极大似然估计 | 可汗学院 / 陈希孺《概率论与数理统计》 |
| **微积分** | 导数、梯度、链式法则、优化 | - |

---

## 📖 机器学习基础（2-3 个月）

### 推荐课程
- **吴恩达《Machine Learning Specialization》** — 最佳入门
- **周志华《机器学习》（西瓜书）** — 中文经典教材

### 核心知识点
- [ ] 监督学习：线性回归、逻辑回归、SVM、决策树
- [ ] 无监督学习：K-Means、PCA、t-SNE
- [ ] 集成学习：随机森林、XGBoost、GBDT
- [ ] 模型评估：交叉验证、过拟合/欠拟合、正则化
- [ ] 特征工程与数据预处理

---

## 📖 深度学习（2-3 个月）

### 推荐课程
- **吴恩达《Deep Learning Specialization》**
- **李沐《动手学深度学习》** — 理论与实践结合

### 核心知识点
- [ ] 神经网络基础与反向传播
- [ ] CNN 卷积神经网络
- [ ] RNN / LSTM 循环神经网络
- [ ] Transformer 架构
- [ ] PyTorch 框架实践

---

## 📖 大语言模型与 LLM（2-3 个月）

### 核心知识点

- [ ] 预训练与微调范式
- [ ] Prompt Engineering 提示工程
- [ ] In-Context Learning 上下文学习
- [ ] Chain-of-Thought 思维链
- [ ] RLHF 基于人类反馈的强化学习
- [ ] 模型压缩：量化、剪枝、蒸馏
