# JVM 内存模型

## 一、运行时数据区

JVM运行时主要分为：

### 1. 堆（Heap）
- 对象实例存储区域
- GC主要管理区域

### 2. 栈（Stack）
- 方法调用栈
- 局部变量

### 3. 方法区（Method Area）
- 类信息
- 常量池
- 静态变量

### 4. 程序计数器
- 当前线程执行位置
